Detailed Description

Utility class for reporting statistics.

Useful for reporting data such as message bus statistics or frame rate and printing it on screen

Reports a name/value pair.

Parameters
nameName of the value to report
valueValue to report

Normally used to report information such as frame rate or key bindings report("FPS", std::to_string(framerate)); report("Space", "Jump");

Once a name/value pair has been reported it will remain in the output even if the value is not updated. To update a value simply report the same name with a new value.
