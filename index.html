xyginext

A 2D Game Engine Framework built around SFML, xyginext implements a scene graph along with an Entity Component System. The source for xyginext is released under the zlib license in the hope that it may be useful for other people - but offers no warranty that it is fit for any particular purpose, even those for which it was specifically written...

Explanations of systems and examples can be found on the xygine wiki.

The older revision of xygine can be found in its own branch.
