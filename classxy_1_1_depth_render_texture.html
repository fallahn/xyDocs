Detailed Description

An off-screen render target with a single texture attached to the depth buffer.

Used internally by the MeshRenderer class for depth mapping the SFML compatible render target has only a single texture attached to the depth buffer. This is useful for techniques such as shadow mapping when depth values need comparing.
