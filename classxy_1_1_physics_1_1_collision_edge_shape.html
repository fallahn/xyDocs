Detailed Description

Creates a 2-sided chain or loop CollisionShape from a series of points.

Useful for outlining static geometry or in some cases where concave polygons are needed

Constructor & Destructor Documentation

CollisionEdgeShape::CollisionEdgeShape ( const std::vector< sf::Vector2f > &  points,
Option  option = Option::None 
)

Creates an edge shape from the given list of points.

If the GhostPoints option is supplied the first and last points will be created as ghost vertices. If the Loop option is supplied then the edges will created as a closed loop.
